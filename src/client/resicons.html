Round Icon Launchers

Use this tool to round corners of icon launchers, crop it, and finally save it as several mipmap sizes into your available mobile application repository.
You can crop icons, drop shadows and apply radius to icon borders.



Create Icon Launchers

Use this tool to create icon launchers from scratch.
You can choose background consist of color or transparent, add an icon on top and send for rounding corners.



Image Editor

Use this tool to create or edit graphical contents of the mobile application resource data such as icon launchers.
